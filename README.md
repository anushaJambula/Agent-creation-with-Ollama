# Agent Learning Repository

## Overview

This repository is a collection of my notes and practice work while learning about agent-based systems.

It includes simple explanations, small experiments, and basic projects as I build my understanding step by step.

## Purpose

* Learn the basics of how agents work
* Practice by building small examples
* Keep track of what I learn over time

## Repository Structure

```
.
├── notes/        # Simple explanations of concepts
├── experiments/  # Small practice tasks
├── projects/     # Basic projects combining ideas
└── reflections/  # What I learned and observed
```

### Notes

Beginner-friendly explanations of topics like:

Small exercises to try out concepts and see how they work in practice.
